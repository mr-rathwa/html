Q.19)What are Logical And Physical Tags in HTML?

Physical Tag:

To specify how characters should be displayed using HTML tags, physical tags are utilised. Only some items may have physical tags applied to them for style purpose. Every physical tag has a defined style, but you may override that style by giving each tag a unique appearance. All physical tags are needs to end with Ending tags .

Example:

  1. < sub > < /sub >
  2. < sup > < /sup >
  3. < i > < /i >
  4. < strike > < /strike >

Logical Tag:

Logical tag allows the browser to display the information in most appropriate manner.

Example:

  1. < abbr > : it is defines as abbreviation of text.
  2. < address > : it defines the information regarding organization.
  3. < strong > : it defines the impotances of text.
  4. < var > : it defines the variable in mathematical equation.

Q.20)What are HTML Entities? With Example.

An HTML entity is a piece of text ("string") that starts with (&) and ends with (;) .HTML entities are mostly used to display reserved characters, and invisible characters.

Examples :

  • & : it used as beginning of entity character.
  • < : it informs beginning of tag.
  • > : it informs Ending of tag.
  •   : It displays non breaking space.
  • ° : It displays degree symbol.
